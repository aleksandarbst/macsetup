## System Preferences

###### Trackpad

* Set Click to `Light`
* Set Tracking speed to notch `7`
* Set Secondary click to `Click in bottom left corner`

###### Finder

In preferences, leave enabled only:
* `Desktop`
* `Downloads`
* `Home`

###### Keyboard
* Set Key Repeat to notch `7`
